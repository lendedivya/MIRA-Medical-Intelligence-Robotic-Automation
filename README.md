# MIRA-Medical-Intelligence-Robotic-Automation
🏥 MIRA — AI-powered health prediction web app using Python Flask, SQLite &amp; Google Gemini AI. Enter patient blood test results and get instant AI-generated health assessments with full CRUD operations.
🚀 Tech Stack
LayerTechnologyBackendPython + FlaskDatabaseSQLiteFrontendHTML + Bootstrap 5AI APIGoogle Gemini 2.0 Flash

✅ Features

CRUD Operations — Create, Read, Update, Delete patient records
AI Remarks — Automatic health assessment via Google Gemini AI
Fallback Mode — Local remarks if AI is unavailable
Validation — Email, date of birth, and numeric field checks
Persistent Storage — SQLite database
